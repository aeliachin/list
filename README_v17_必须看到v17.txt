家庭食谱 v17 可见版本确认版

目的：
页面顶部会显示 “共享家庭菜单 · v17” 和 “v17 新版”。
如果上传后页面上看不到 v17，说明浏览器/GitHub Pages 没有读取你刚替换的文件。

上传：
把本包里的所有文件上传到 GitHub Pages 正在使用的根目录，并覆盖同名文件。

上传后打开：
https://aeliachin.github.io/?v=17
或
https://aeliachin.github.io/xiaofei/?v=17

如果仍看不到 v17：
1. 检查是否上传到了正确仓库/正确文件夹。
2. 检查 GitHub Pages 设置里 Branch 和 Folder。
3. 等 2-5 分钟。
4. 手机 Safari 关闭网页重新打开，或用电脑无痕窗口打开。
