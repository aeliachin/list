# 萌宠食谱

这是“萌宠食谱”v4 登录诊断增强版。

上传到 GitHub Pages 仓库根目录的文件：
- index.html
- style.css
- app.js
- site.webmanifest
- icon-192.png
- icon-512.png
- apple-touch-icon.png

Supabase 数据库脚本：
- supabase_recipe_app.sql

重要：
1. 打开 app.js 顶部，把 SUPABASE_URL 和 SUPABASE_ANON_KEY 改成你自己的 Supabase 项目信息。
2. GitHub Pages 上传后，用 ?v=4 或 ?v=5 打开，避免浏览器缓存旧版 app.js。
3. 不要上传 ZIP 本身，要上传 ZIP 解压后的文件。
